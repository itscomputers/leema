leema
======

![https://travis-ci.org/mdg/leema.svg?branch=master]

Leema is a concurrent, error-tolerant, functional programming language.

## Build

Dependencies:
* lemon_rust submodule
* flex

Build the leema vm by running `make`

## Tests

All tests
```
make test
```

Unit tests only
```
make unit
```

Integration tests only
```
make T
```
